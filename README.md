# deathPunishment

<h2>功能列表：</h2>

<li>随机掉落n件物品</li>
<li>掉落经验值</li>
<li>显示死亡标题</li>
<li>重生扣生命/饥饿</li>
<li>重生时给予药水效果</li>

<h2>With this plugin,you can</h2>

<li>Drop items randomly</li>
<li>Drop experience</li>
<li>Show death titles</li>
<li>Limit respawn health and/or food level</li>
<li>Cast potion effects on respawn</li>
